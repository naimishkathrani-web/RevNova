Day 1: GitHub Actions Setup

Developer 3 - DevOps & QA | Duration: ~6-8 hours

1 Install Git

What you need to do: Install version control system

  1. Download from https://git-scm.com/download/win
  2. Run the installer, keep all default settings
  3. Restart your terminal

Verify installation:

git --version

You should see: git version 2.x.x

2 Install Node.js

What you need to do: Install Node.js for running tests

  1. Go to https://nodejs.org/
  2. Download Node.js 20 LTS
  3. Run installer, click "Next" through all steps

Verify installation:

node --version npm --version

3 Install Docker Desktop

What you need to do: Install container platform

  1. Go to https://www.docker.com/products/docker-desktop/
  2. Download Docker Desktop for Windows
  3. Run installer
  4. Restart computer when prompted
  5. Start Docker Desktop from Start menu
  6. Wait for "Docker Desktop is running" in system tray

Verify Docker is running:

docker --version docker ps

You should see Docker version and an empty container list.

üí° Docker Desktop must be running! You'll see a whale icon in the system tray when it's ready.

4 Clone Repository

What you need to do: Get the RevNova code

Open PowerShell and run:

mkdir C:\Dev cd C:\Dev git clone https://github.com/naimishkathrani-web/RevNova.git RevNovaRepository cd RevNovaRepository

Install VS Code from https://code.visualstudio.com/

Open the repository:

code .

5 Create GitHub Actions Folder

What you need to do: Set up CI/CD directory structure

In VS Code terminal, run:

mkdir .github mkdir .github\workflows type nul > .github\workflows\.gitkeep
üí° What is .github/workflows? This folder holds YAML files that define automated workflows (CI/CD pipelines).

6 Learn YAML Basics

What you need to do: Understand YAML syntax for workflows

YAML Key Concepts:

  • Indentation: Use 2 spaces (NOT tabs!)
  • Key-value: name: My Workflow
  • Lists: Use dash (-) for list items
  • Multiline: Use | or > for long text

Example YAML:

name: CI Pipeline on: push: branches: [main] pull_request: branches: [main] jobs: test: runs-on: ubuntu-latest steps: - name: Checkout code uses: actions/checkout@v3 - name: Run tests run: npm test
üí° Key parts: on: = when to run, jobs: = what to do, steps: = individual tasks.

7 Verify Setup

What you need to do: Check all tools are ready

Run these commands:

git --version node --version docker --version docker ps

Check folder structure:

dir .github\workflows
  • Git installed and working
  • Node.js and npm installed
  • Docker Desktop running
  • Repository cloned
  • .github/workflows folder created
  • Understand basic YAML syntax
