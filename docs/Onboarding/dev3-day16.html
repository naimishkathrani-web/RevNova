Day 16: Deploy to Dev Workflow

Developer 3 — DevOps & QA

Objective

Create GitHub Actions workflow to automatically deploy to dev environment on main branch push.

Tasks

  1. Create .github/workflows/deploy-dev.yml
  2. Build and push Docker images to registry
  3. Deploy to dev server (cloud or VM)
  4. Run smoke tests after deployment
