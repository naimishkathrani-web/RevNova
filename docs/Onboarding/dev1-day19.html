Day 19: ETL Job Processing

Developer 1 — Backend & Database

Objective

Implement worker logic to extract source data, transform via mappings, and stage for load.

Tasks

  1. Build ETL worker that reads from source connection

  3. Write transformed data to staging tables (STG1/STG2)
  4. Update job progress and emit status events
