Day 21: Migration Execution API

Developer 1 — Backend & Database

Objective

Build the final migration execution API to load transformed data from staging to target.

Tasks

  1. Create POST /api/migrations/:id/execute endpoint
  2. Read transformed data from staging tables
  3. Load data to target system via API or direct DB insert
  4. Track execution progress and emit real-time status updates
