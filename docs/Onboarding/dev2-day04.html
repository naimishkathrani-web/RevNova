Developer 2 - Frontend & React | Duration: ~6-8 hours

1 Install Zustand

What you need to do: Add state management library

In terminal, run:

cd frontend npm install zustand

You should see:

added 1 package
üí° Why Zustand? Simpler than Redux, smaller than MobX. Perfect for beginners! Stores app-wide data like user settings, migration progress, etc.

2 Create Wizard Store

What you need to do: Build store to track wizard progress

Create folder and file:

mkdir src\store type nul > src\store\wizardStore.ts

Add this code to frontend/src/store/wizardStore.ts:

import { create } from 'zustand'; interface WizardState { currentStep: number; completedSteps: number[]; migrationId: string | null; setCurrentStep: (step: number) => void; markStepComplete: (step: number) => void; setMigrationId: (id: string) => void; resetWizard: () => void; } export const useWizardStore = create((set) => ({ currentStep: 1, completedSteps: [], migrationId: null, setCurrentStep: (step) => set({ currentStep: step }), markStepComplete: (step) => set((state) => ({ completedSteps: state.completedSteps.includes(step) ? state.completedSteps : [...state.completedSteps, step] })), setMigrationId: (id) => set({ migrationId: id }), resetWizard: () => set({ currentStep: 1, completedSteps: [], migrationId: null }) }));

3 Create Connection Store

What you need to do: Build store for Salesforce connection data

Create file: frontend/src/store/connectionStore.ts

import { create } from 'zustand'; interface ConnectionData { instanceUrl: string; username: string; apiVersion: string; isConnected: boolean; } interface ConnectionState { source: ConnectionData; target: ConnectionData; setSourceConnection: (data: Partial) => void; setTargetConnection: (data: Partial) => void; resetConnections: () => void; } const defaultConnection: ConnectionData = { instanceUrl: '', username: '', apiVersion: 'v58.0', isConnected: false }; export const useConnectionStore = create((set) => ({ source: defaultConnection, target: defaultConnection, setSourceConnection: (data) => set((state) => ({ source: { ...state.source, ...data } })), setTargetConnection: (data) => set((state) => ({ target: { ...state.target, ...data } })), resetConnections: () => set({ source: defaultConnection, target: defaultConnection }) }));
