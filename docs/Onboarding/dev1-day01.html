Day 1: Workstation Setup

Developer 1 - Backend & Database | Duration: ~6-8 hours

1 Install Node.js

What you need to do: Download and install Node.js version 20

  1. Go to https://nodejs.org
  2. Click "Download Node.js (LTS)" - get version 20.x
  3. Run the installer
  4. Click "Next" through all steps (keep default settings)
  5. Wait for installation to finish

Check if it worked:

node --version
npm --version

You should see version numbers like:

v20.x.x
10.x.x
üí° Tip: If you see "command not found", close and reopen your terminal (PowerShell or Command Prompt)

2 Install Git

What you need to do: Install Git to download and manage code

  1. Go to https://git-scm.com/download/win
  2. Download the installer
  3. Run the installer
  4. Click "Next" through all steps (keep default settings)

Check if it worked:

git --version

You should see something like:

git version 2.x.x

3 Install VS Code

What you need to do: Install Visual Studio Code (your code editor)

  1. Go to https://code.visualstudio.com
  2. Click "Download for Windows"
  3. Run the installer
  4. ‚úÖ Check "Add to PATH" during installation
  5. ‚úÖ Check "Create Desktop Icon"
  6. Click "Install"

Open VS Code: Double-click the VS Code icon on your desktop

4 Install VS Code Extensions

What you need to do: Add helpful tools to VS Code

  1. Open VS Code
  2. Click the Extensions icon on the left sidebar (looks like 4 squares)
  3. Search for and install these extensions (click "Install" button for each):
  • GitHub Copilot
  • GitHub Copilot Chat
  • ESLint
  • Prettier - Code formatter
  • GitLens
  • PostgreSQL (by Chris Kolkman)
  • Thunder Client (for testing APIs)
üí° Tip: After installing each extension, you'll see a "Reload" or "Restart" button. Click it after installing all extensions.

5 Clone the Repository

What you need to do: Download the RevNova code to your computer

  1. Open VS Code
  2. Press Ctrl+` (backtick) to open the terminal inside VS Code
  3. Type this command and press Enter:
cd C:\Dev

If you see "cannot find the path", create the folder first:

mkdir C:\Dev
cd C:\Dev

Now clone the repository:

git clone https://github.com/naimishkathrani-web/RevNova-Dev.git RevNovaRepository

This will download all the code. Wait for it to finish (might take 1-2 minutes).

Open the project:

  1. In VS Code, click File ‚Üí Open Folder
  2. Navigate to C:\Dev\RevNovaRepository
  3. Click "Select Folder"

6 Install Project Dependencies

What you need to do: Install all the libraries that RevNova needs

In the VS Code terminal, type:

cd backend
npm install

This will download all the required packages. It might take 3-5 minutes. You'll see lots of text scrolling - that's normal!

üí° Tip: If you see any warnings (yellow text), that's okay. Only worry about errors (red text).

Wait until you see something like:

added 847 packages in 4m

7 Verify Everything Works

Final check: Make sure everything is installed correctly

  • Node.js installed (run: node --version)
  • Git installed (run: git --version)
  • VS Code open with RevNova project
  • Extensions installed (check Extensions panel in VS Code)
  • Repository cloned (you can see folders like "backend", "frontend", "docs")
  • Backend dependencies installed (you have a "node_modules" folder in backend)

Test that backend runs:

npm run dev

You should see:

Server running on port 3000

Press Ctrl+C to stop the server.
