Day 5: First API Endpoint

Developer 1 - Backend & Database | Duration: ~6-8 hours

1 Setup Backend Project Structure

What you need to do: Create folders for your Express API

Open terminal and run:

cd C:\Dev\RevNovaRepository\backend mkdir src\routes src\controllers src\services type nul > src\index.ts
üí° Project structure: routes = URL endpoints, controllers = handle requests, services = business logic, index.ts = app startup.

2 Install Backend Dependencies

What you need to do: Install Express and TypeScript packages

Run these commands:

cd C:\Dev\RevNovaRepository\backend npm install express cors dotenv pg npm install --save-dev typescript @types/node @types/express @types/cors ts-node-dev

Initialize TypeScript:

npx tsc --init

You should see: "Created a new tsconfig.json"

3 Create Your First API Endpoint

What you need to do: Create a health check endpoint

Create file: backend/src/index.ts

import express, { Request, Response } from 'express'; import cors from 'cors'; const app = express(); const PORT = process.env.PORT || 3000; // Middleware app.use(cors()); app.use(express.json()); // Health check endpoint app.get('/api/v1/health', (req: Request, res: Response) => { res.status(200).json({ status: 'ok', message: 'RevNova API is running!', timestamp: new Date().toISOString(), database: 'connected' }); }); // Start server app.listen(PORT, () => { console.log(`üöÄ Server running on http://localhost:${PORT}`); console.log(`üìã Health check: http://localhost:${PORT}/api/v1/health`); }); export default app;

4 Add npm Scripts

What you need to do: Update package.json to run the server

Open backend/package.json and add these scripts:

"scripts": { "dev": "ts-node-dev --respawn --transpile-only src/index.ts", "build": "tsc", "start": "node dist/index.js" }
üí° What do these do? "dev" runs the server and auto-restarts on file changes. "build" compiles TypeScript to JavaScript. "start" runs the compiled code.

5 Test Your API

What you need to do: Start the server and test the endpoint

Start the development server:

npm run dev

You should see:

üöÄ Server running on http://localhost:3000 üìã Health check: http://localhost:3000/api/v1/health

In VS Code, install Thunder Client extension (if not already installed)

Test the endpoint:

  1. Click Thunder Client icon in sidebar
  2. Click "New Request"
  3. Set URL to: http://localhost:3000/api/v1/health
  4. Click "Send"

You should see:

{ "status": "ok", "message": "RevNova API is running!", "timestamp": "2025-01-...", "database": "connected" }

6 Commit Week 1 Work

What you need to do: Save your progress to Git

Stop the server (press Ctrl+C in terminal)

Commit your work:

git add . git commit -m "feat: Week 1 complete - database schema and first API endpoint" git log --oneline -5

You should see your commit in the list!

üí° Don't push yet! We'll push all Week 1 work together after all 3 developers complete their week.
