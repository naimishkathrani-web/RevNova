Day 19: Deploy to Test Workflow

Developer 3 — DevOps & QA

Objective

Create separate workflow for deploying to test/staging environment.

Tasks

  1. Create deploy-test.yml workflow triggered by tag or manual dispatch

  3. Run full E2E test suite against test environment
  4. Generate deployment report
