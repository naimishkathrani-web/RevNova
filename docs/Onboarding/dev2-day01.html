‚úÖ Day 1: React Setup [COMPLETED]

Developer 2 - Frontend & React | Duration: ~6-8 hours

1 Install Node.js

What you need to do: Install Node.js (same as Developer 1)

  1. Go to https://nodejs.org/
  2. Download Node.js 20 LTS (Long Term Support)
  3. Run the installer, click "Next" through all steps
  4. Check "Automatically install necessary tools" checkbox

Verify installation:

node --version npm --version

You should see version numbers like v20.x.x and 10.x.x

2 Install Git and VS Code

What you need to do: Install development tools

Install Git:

  1. Download from https://git-scm.com/download/win
  2. Run installer, keep all defaults

Install VS Code:

  1. Download from https://code.visualstudio.com/
  2. Run installer
  3. ‚úÖ Check "Add to PATH"
  4. ‚úÖ Check "Create desktop icon"

3 Install VS Code Extensions

What you need to do: Install React development extensions

Open VS Code and install these extensions (Ctrl+Shift+X):

  • ES7+ React/Redux/React-Native snippets
  • ESLint
  • Prettier - Code formatter
  • Tailwind CSS IntelliSense
  • Auto Rename Tag
  • GitLens

4 Clone Repository

What you need to do: Get the RevNova code on your computer

Open terminal in VS Code (Ctrl+`) and run:

mkdir C:\Dev cd C:\Dev git clone https://github.com/naimishkathrani-web/RevNova.git RevNovaRepository cd RevNovaRepository

Open the folder in VS Code: File ‚Üí Open Folder ‚Üí select C:\Dev\RevNovaRepository

5 Install Frontend Dependencies

What you need to do: Install React and all packages

Navigate to frontend folder and install:

cd frontend npm install

This will take 2-3 minutes. You should see:

added 1234 packages in 2m
üí° What did this install? React 18, TypeScript, Vite (build tool), TailwindCSS, and development tools.

6 Start Development Server

What you need to do: Run the React app

Start the dev server:

npm run dev

You should see:

VITE v5.x.x ready in 500 ms ‚ûú Local: http://localhost:5173/ ‚ûú Network: use --host to expose

Open your browser and go to http://localhost:5173/

You should see the React + Vite welcome page! üéâ

7 Verify Everything Works

What you need to do: Test that hot reload works

With the server still running, open frontend/src/App.tsx

Change the title text to "RevNova Migration Tool" and save (Ctrl+S)

Your browser should automatically refresh and show the new text!

Press Ctrl+C in terminal to stop the server.

  • Server starts without errors
  • Browser shows React app at localhost:5173
  • Hot reload works when you edit files
  • No console errors in browser (press F12)
