Day 1: React & Vite Setup

Developer 2: Frontend & React | Duration: ~6-8 hours

üìã Objective

Set up your frontend development environment with React and Vite. By the end of this day, you'll have a modern React application running locally with proper project structure and routing configured.

What You'll Accomplish:

  • Install Node.js LTS (v18+) and npm
  • Create a new Vite + React project
  • Set up project folder structure
  • Install essential dependencies (React Router, Axios, Zustand)
  • Configure basic routing
  • Create a layout component
  • Test the development server

üì¶ Step 4: Install Essential Dependencies

Install React Router for navigation:

npm install react-router-dom

Install Axios for API calls:

npm install axios

Install Zustand for state management:

npm install zustand

Install additional utilities:

npm install clsx

Why these libraries?

  • React Router - Client-side routing
  • Axios - Promise-based HTTP client
  • Zustand - Lightweight state management (simpler than Redux)
  • clsx - Conditional className utility

‚úÖ Checkpoint: All dependencies installed successfully

üó∫Ô∏è Step 5: Configure React Router

File: src/App.tsx

import { BrowserRouter, Routes, Route } from 'react-router-dom';
import Layout from './components/layout/Layout';
import Home from './pages/Home';
import Dashboard from './pages/Dashboard';
import NewMigration from './pages/NewMigration';

function App() {
  return (
    <BrowserRouter>
      <Routes>
        <Route path="/" element={<Layout />}>
          <Route index element={<Home />} />
          <Route path="dashboard" element={<Dashboard />} />
          <Route path="new-migration" element={<NewMigration />} />
        </Route>
      </Routes>
    </BrowserRouter>
  );
}

export default App;
