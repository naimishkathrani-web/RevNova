Day 2: Component Library

Developer 2 - Frontend & React | Duration: ~6-8 hours

1 Create Components Folder Structure

What you need to do: Set up folders for UI components

In VS Code terminal, run:

cd frontend\src mkdir components components\ui components\layout type nul > components\index.ts
üí° Folder structure: ui = reusable components (Button, Input), layout = page structure components (Header, Sidebar).

Create file: frontend/src/components/ui/Button.tsx

5 Export Components

What you need to do: Create a barrel export file

Update file: frontend/src/components/index.ts
