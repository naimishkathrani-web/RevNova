Developer 2: Frontend & React | Duration: ~6-8 hours

Enhance the schema analysis API to fetch detailed field metadata, implement job status polling, and add Redis caching for performance optimization.

üì¶ Step 1: Install Redis Client

Install the Redis client for Node.js to enable caching:

cd backend
npm install redis

Create a Redis client configuration:

File: backend/src/config/redis.ts

import { createClient } from 'redis';

const redisClient = createClient({
  url: process.env.REDIS_URL || 'redis://localhost:6379'
});

redisClient.on('error', (err) => console.error('Redis Client Error', err));

export const connectRedis = async () => {
  await redisClient.connect();
  console.log('Redis connected');
};

export default redisClient;

üîÑ Step 2: Create Job Status Polling Endpoint

File: backend/src/routes/analyze.routes.ts

router.get('/projects/:id/analyze/:jobId', async (req, res) => {
  try {
    const { id, jobId } = req.params;
    
    // Check Redis for job status
    const jobData = await redisClient.get(`job:${jobId}`);
    
    if (!jobData) {
      return res.status(404).json({ error: 'Job not found' });
    }
    
    const job = JSON.parse(jobData);
    res.json({
      jobId: job.id,
      status: job.status, // 'pending', 'running', 'completed', 'failed'
      progress: job.progress,
      result: job.result || null,
      error: job.error || null
    });
  } catch (error) {
    res.status(500).json({ error: error.message });
  }
});

üìä Step 3: Enhance Field Metadata

Update the schema analysis to include detailed field information:

const getDetailedFieldMetadata = (field: any) => {
  return {
    name: field.name,
    label: field.label,
    type: field.type,
    length: field.length,
    precision: field.precision,
    scale: field.scale,
    required: !field.nillable,
    unique: field.unique,
    externalId: field.externalId,
    defaultValue: field.defaultValue,
    picklistValues: field.picklistValues || [],
    referenceTo: field.referenceTo || [],
    relationshipName: field.relationshipName,
    calculated: field.calculated,
    customField: field.custom
  };
};
